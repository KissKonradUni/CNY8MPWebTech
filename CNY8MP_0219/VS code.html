VS Code-ban az alábbi dolgokat használom kezdés előtt:

  • Belépek GitHub-ba, hogy a beállításokat szinkronizálja.
  • Letöltöm a Visual Studio Keymapot, megszokás miatt.
  • Letöltöm az IntelliSense bővítményeket Javascript és Typescript nyelvekhez, illetve a HTML CSS Support-ot.
  • Letöltöm a Live Server, a Doxygen Doc. Generator, Prettier, és az Error Lens bővítményeket.
  • Letöltöm a Material Icon packet, a Darcula kód színt, és a CodeSnap bővítményt.
  • Minden projektet minimum egy lokális GitHub repositoryval kezdek.
